I am reaching out to you regarding a client of mine: Gizmox. (http://www.visualwebgui.com). Visual WebGUI is their popular open source tool.

I was interested to see if you are involved in any projects that are modernizing and upgrading legacy client server, web and older applications into HTML5-based web and mobile applications.

Gizmox provides tools that automate the modernization and upgrade process for "legacy" Windows applications and extends local applications to native HTML5 Web, Cloud and Mobile deployments - all without costly rewrites and application virtualization.

=> Cost Effective: Typically our costs range between $.35 to $.50 per Line of Code (as compared to $6 - $26 per LOC for a code rewrite).
=> Rapid Execution: In a recent POC we moved 90K LOC from VB6 to .Net in 3 Weeks.

Gizmox provides an easier way to move and upgrade existing applications, one that reduces migration risk, minimizes costs, and accelerates implementation time.
